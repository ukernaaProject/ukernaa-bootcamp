Ukernaa Web Design Bootcamp

HTML 5

A web page's structure is created using HTML5, a markup language. We will assist you in learning the fundamentals of HTML5 through this bootcamp. We'll go from a beginner's level to a point where you can use HTML5 to build stunning web page architectures.

CSS 3

CSS is a language for creating style sheets that describe how a document written in a markup language like HTML should be presented. We'll demonstrate how to visually appealingly style an HTML5 document.

JavaScript

Along with HTML and CSS, the programming language JavaScript is one of the foundational elements of the World Wide Web. 98% of websites will utilise JavaScript for client-side webpage behaviour by 2022.

Git

Git is a free and open source application for distributed version control that tracks changes in any group of files. It is typically used to organise the efforts of programmers who are working together to build source code for software. We'll run through some of git's fundamental usage cases.

GitHub

Software development and version control using Git are both supported by the website GitHub. It offers each project wikis, access control, bug tracking, software feature requests, continuous integration, and Git's distributed version control.

About this Bootcamp

We present to you a web design and deployment bootcamp from 26th September 2022 to 28th September 2022 which will provide all the attendees a perfect entrypoint into fullstack development at the oh-so appealing and enticing cost of ZERO. The contents of the bootcamp will be as follows:

  • HTML5
  • CSS3
  • Javascript (ES6)
  • Git
  • GitHub
  • Web Deployment
